Dashboard visualization of PhDs data in the US with streamlit

Published: Fri 06 November 2020
By Fan Yang

In posts.

tags: dashboard data visualization streamlit

Introduction

In this post, I will make a dashboard visualization using data of PhDs awarded in the US with help of steamlit. The data includes tables of detailed data on the demographic characteristics, educational history, sources of financial support, and postgraduation plans of doctorate recipients. I will choose two aspects, numbers of doctorate recipients for broad field of study and age distribution of doctorate recipients, to explore the data.

Using streamlit, I can create an interactive app for data or model. Streamlit's UI allows me to work in a fast interactive loop: first write some code in .py file, save it, run the app and review results, write more code, and so on.

Streamlit setup

# Install streamlit
pip install streamlit

Create the Streamlit app

  1. Create a new Python file named first_app.py, then open it with IDE or text editor.
  2. Import Streamlit
import streamlit as st
import numpy as np
import pandas as pd
  1. Run the app. A new tab will open in the default browser.
streamlit run first_app.py
  1. To kill the app, type Ctrl+c in the terminal.

Dashboard interaction

For chart I, I can use sidebox to view the number of doctorate recipients by selecting the specific broad field of study. The result will be displayed in bar chart. To show citizenship status, just click that button on the left side. The default chart is stacked bar chart. Click the button saying 'show grouped citizenship status', the chart will be converted to grouped bar chart.

For chart II, I use pie chart to show the age distribution of doctorate recipients. Use different option, such as broad field of study and sex, to view or compare the distribution.
