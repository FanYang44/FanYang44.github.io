My very first blog

The article is written with Markdown. The idea is firstly, edit the .md file in Typora which is a very powerful editor for many kinds of programming language. Secondly, I use the Pelican, a static site generator powered by Python, to convert the .md file into .html file. Before the last step, I am supposed to ensure that the my own github personal page is ready to use. Finally, the .html file should be able to be pushed to my github website as a part of my blog.

Steps

  • Create a working directory for my blog.
mkdir blog
cd blog
  • Use virtualenv to create an env.
virtualenv env
source env/bin/activate # Activate the env
  • Use pip to install Pelican and Markdown.
pip install pelican
pip install markdown
  • Create a pelican skeleton via pelican-quickstart command which help me set up pelican config by asking multiple questions.
pelican-quickstart
  • Move my .md file into content directory. One thing to mention is that the .md file must have a header to allow pelican to recognize.
# One example of the header from pelican documentation

Title: My First Review
Date: 2010-12-03 10:20
Category: Review
  • Have a look at the local output.
# Under blog/ directory

pelican content # Generate static site under output/ directory  according to content 

# Under output/ directory
python -m pelican.server

If there is no error reported, go to 127.0.0.1:8000 via web browser. Type in ctrl + c to terminate this service.

  • Upload the genetated static site to github pages.
  • First set up a github repository with username.github.io as Repository name.
  • Under output/ directory, add/commit changes.
git init
git add .
git commit -m "first commit"
git remote add origin git@github.com:uesrname/username.github.io.git
git push -u origin master
