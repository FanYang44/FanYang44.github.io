My very first blog

The article is written with markdown. The idea is firstly, finish the markdown in Typora which is a very powerful editor for many kinds of programming language. Then, I use the Pelican, a static site generator powered by python, to convert the written .md file into .html file. Before the last step, I am supposed to ensure that the my own github personal page is ready to use. Finally, the .html file should be able to be pushed to my git website as part of my blog.
